# 🔍 Новые функции сканирования

## 📋 Добавленные команды

### 🔍 Принудительное сканирование

#### `/scan` - Сканирование всех ссылок
Принудительно сканирует все ваши отслеживаемые ссылки и показывает результаты в реальном времени.

**Пример использования:**
```
/scan
```

**Результат:**
```
🔍 Начинаю сканирование 3 ссылок...
📡 Сканирование 1/3: https://example.com/events...
📊 Результат сканирования:
🔗 https://example.com/events
📅 Найдено событий: 5
🆕 Новых событий: 2
🔄 Последняя проверка: 04.10.2025 15:30:45
```

#### `/scan <номер>` - Сканирование конкретной ссылки
Сканирует только указанную ссылку по номеру из списка.

**Пример использования:**
```
/scan 1
```

**Результат:**
```
🔍 Сканирование ссылки 1...
📊 Результат сканирования:
🔗 https://example.com/events
📅 Найдено событий: 5
🆕 Новых событий: 0
🔄 Последняя проверка: 04.10.2025 15:30:45
```

### 🧪 Тестирование парсинга

#### `/test <ссылка>` - Тестирование ссылки
Тестирует парсинг любой ссылки без добавления её в мониторинг.

**Пример использования:**
```
/test https://example.com/events
```

**Результат:**
```
🧪 Тестирование парсинга ссылки...
🔗 https://example.com/events
🧪 Результат тестирования:
🔗 https://example.com/events
📅 Найдено событий: 5
⏰ Время тестирования: 04.10.2025 15:30:45

📋 Найденные события:
1. Концерт группы "Рок-звезды"
   📆 15.12.2024 20:00
   📍 Концертный зал "Мегаполис"
   💰 от 1500 руб
   🔗 https://example.com/event/123
```

### 📊 Улучшенный статус

#### `/status` - Детальная информация
Показывает подробную информацию о всех ваших ссылках.

**Результат:**
```
📊 Статус мониторинга:

🔗 Всего отслеживаемых ссылок: 3
👤 Ваших ссылок: 3
⏰ Интервал проверки: 10 минут
🔄 Текущее время: 04.10.2025 15:30:45

✅ Мониторинг активен

📋 Ваши ссылки:
1. https://example.com/events...
   Последняя проверка: 04.10.2025 15:20:30
   Найдено событий: 5

2. https://music-venue.ru/concerts...
   Последняя проверка: 04.10.2025 15:15:45
   Найдено событий: 3
```

### 📋 Логи проверок

#### `/logs` - Последние проверки
Показывает когда последний раз проверялись ваши ссылки.

**Результат:**
```
📋 Последние проверки ваших ссылок:

1. https://example.com/events...
   📅 Последняя проверка: 04.10.2025 15:20:30
   ⏰ 10 минут назад
   📊 Найдено событий: 5
   🔄 Статус: ✅ Активен

2. https://music-venue.ru/concerts...
   📅 Последняя проверка: 04.10.2025 15:15:45
   ⏰ 15 минут назад
   📊 Найдено событий: 3
   🔄 Статус: ✅ Активен
```

## 🎯 Преимущества новых функций

### ✅ Контроль над мониторингом
- **Принудительное сканирование** - можете проверить ссылки в любое время
- **Тестирование ссылок** - проверьте парсинг перед добавлением в мониторинг
- **Детальная статистика** - видите когда и что было найдено

### ✅ Диагностика проблем
- **Проверка статуса** - убедитесь что бот работает
- **Логи проверок** - видите когда последний раз проверялись ссылки
- **Тестирование парсинга** - проверьте работает ли парсинг на конкретном сайте

### ✅ Прозрачность работы
- **Реальное время** - видите результаты сканирования сразу
- **Подробная информация** - количество событий, новых событий, время проверки
- **Статус активности** - знаете активен ли мониторинг

## 🚀 Как использовать

### 1. Добавьте ссылку для мониторинга
```
/add https://example.com/events
```

### 2. Протестируйте парсинг
```
/test https://example.com/events
```

### 3. Проверьте статус
```
/status
```

### 4. Принудительно просканируйте
```
/scan
```

### 5. Посмотрите логи
```
/logs
```

## 💡 Советы по использованию

- **Используйте `/test`** перед добавлением новой ссылки
- **Проверяйте `/status`** чтобы убедиться что мониторинг активен
- **Используйте `/scan`** если хотите проверить ссылки немедленно
- **Смотрите `/logs`** чтобы понять когда последний раз проверялись ссылки

## 🔧 Технические детали

- Все команды работают **асинхронно**
- **Безопасность** - каждый пользователь видит только свои ссылки
- **Производительность** - пауза между сканированиями для избежания перегрузки
- **Логирование** - все действия записываются в логи
- **Обработка ошибок** - подробные сообщения об ошибках
