# ND

Небольшой консольный бот-консьерж.

## Как запустить

1. Убедитесь, что у вас установлен Python 3.9 или новее.
2. В терминале перейдите в корень проекта.
3. (Необязательно) Задайте переменные окружения, чтобы изменить имя и приветствие бота:
   ```powershell
   set BOT_NAME=Jarvis
   set BOT_GREETING=Привет!
   ```
4. Выполните команду:
   ```bash
   python -m bot.main
   ```

При запуске бот выводит приветствие и приглашение ввода. Введите `help`, чтобы увидеть список доступных команд, или `exit`, чтобы завершить работу.

## Структура проекта

- `bot/config.py` — загрузка настроек из переменных окружения.
- `bot/app.py` — логика команд и цикл обработки сообщений.
- `bot/main.py` — точка входа, которая создаёт `Settings` и запускает `BotApp`.
