# 🎵 Телеграм-бот для мониторинга концертных площадок

Бот для автоматического мониторинга сайтов концертных площадок и уведомления о новых событиях.

## 🚀 Возможности

- ✅ Мониторинг веб-страниц каждые 10 минут
- ✅ Уведомления только при появлении новых событий
- ✅ Автоматическое извлечение информации о событиях:
  - Название события
  - Дата проведения
  - Ссылка на событие
  - Афиша (изображение)
  - Место проведения
  - Цена билетов
- ✅ Поддержка множественных ссылок для мониторинга
- ✅ Персональные настройки для каждого пользователя

## 📋 Команды бота

- `/start` - Начать работу с ботом
- `/help` - Показать справку
- `/add <ссылка>` - Добавить ссылку для мониторинга
- `/list` - Показать все отслеживаемые ссылки
- `/remove <номер>` - Удалить ссылку из мониторинга
- `/status` - Показать статус мониторинга

## ⚙️ Технические детали

### Парсинг событий

Бот использует универсальные селекторы для поиска событий на различных сайтах:
- `.event`, `.concert`, `.show`, `.performance`
- `[class*="event"]`, `[class*="concert"]`
- `.card`, `.item`, `.poster`

### Извлечение данных

Для каждого события бот извлекает:
- **Заголовок**: из тегов `h1-h4`, `.title`, `.name`
- **Дата**: из `.date`, `.time`, `.datetime`
- **Ссылка**: из тега `<a>`
- **Изображение**: из тега `<img>`
- **Место**: из `.venue`, `.place`, `.location`
- **Цена**: из `.price`, `.cost`, `.ticket`

### Хранение данных

- `monitored_urls.json` - отслеживаемые ссылки и их состояние
- `events_cache.json` - кэш событий для определения изменений

## 🔧 Настройка

### Изменение интервала мониторинга

В файле `concert_monitor_bot.py` найдите строку:
```python
await asyncio.sleep(600)  # 10 минут
```

Измените значение (в секундах):
- 300 = 5 минут
- 600 = 10 минут
- 1800 = 30 минут

### Добавление новых селекторов

В методе `parse_events_from_page()` добавьте новые селекторы в список `event_selectors`:
```python
event_selectors = [
    '.event', '.concert', '.show', '.performance',
    '.your-custom-selector',  # Добавьте здесь
    # ...
]
```

## 🐛 Устранение неполадок

### Бот не отвечает
- Проверьте правильность токена
- Убедитесь, что бот запущен
- Проверьте логи в консоли

### События не парсятся
- Проверьте доступность сайта
- Убедитесь, что сайт не блокирует ботов
- Добавьте специфичные селекторы для сайта

### Уведомления не приходят
- Проверьте, что ссылка добавлена в мониторинг
- Убедитесь, что на сайте есть новые события
- Проверьте логи на наличие ошибок
