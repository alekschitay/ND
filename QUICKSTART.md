# Быстрый старт Concert Monitor Bot 🚀

## Установка за 3 шага

### 1. Установка зависимостей
```bash
python install.py
```

### 2. Настройка токена
1. Получите токен у [@BotFather](https://t.me/BotFather)
2. Отредактируйте файл `.env`:
```env
BOT_TOKEN=ваш_токен_здесь
```

### 3. Запуск
```bash
python run.py
```

## Первое использование

1. Найдите вашего бота в Telegram
2. Отправьте `/start`
3. Добавьте страницу командой `/add` или просто отправьте ссылку
4. Бот начнет мониторинг!

## Примеры ссылок

- https://kudago.com/moscow/concerts/
- https://afisha.ru/moscow/concert/
- https://concert.ru/events/

## Команды

- `/add` - добавить страницу
- `/list` - показать отслеживаемые страницы
- `/test <url>` - протестировать парсинг
- `/sites` - поддерживаемые сайты
- `/help` - справка

## Остановка

Нажмите `Ctrl+C` в терминале для остановки бота.
