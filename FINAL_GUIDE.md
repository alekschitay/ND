1. В разделе **"➕ Добавить сайт"**
2. Заполните форму:

3. Нажмите **"✅ Добавить сайт"**

### 3. **Настройте уведомления:**
1. В разделе **"⚙️ Настройки"**
2. Добавьте Telegram Bot Token и Chat ID

## 🔧 **КАК ОПРЕДЕЛИТЬ CSS СЕЛЕКТОР:**

### **Простой способ:**
1. Откройте сайт в браузере
2. Нажмите **F12** (инструменты разработчика)
3. Найдите элемент с событием
4. Щелкните правой кнопкой → **"Копировать"** → **"Копировать селектор"**
5. Вставьте в поле "CSS селектор"

### **Примеры селекторов:**
- `.event-card` - элементы с классом event-card
- `.concert-item` - элементы с классом concert-item
- `div.event` - div элементы с классом event
- `[class*="event"]` - элементы содержащие "event" в классе

## 📱 **НАСТРОЙКА TELEGRAM БОТА:**

### **Создание бота:**
1. Напишите [@BotFather](https://t.me/botfather)
2. Отправьте `/newbot`
3. Введите название: "Concert Monitor Bot"
4. Введите username: "concert_monitor_bot"
5. Сохраните токен

### **Получение Chat ID:**
1. Напишите боту `/start`
2. Перейдите: `https://api.telegram.org/bot<YOUR_BOT_TOKEN>/getUpdates`
3. Найдите `"chat":{"id":123456789}`
4. Сохраните ID (число без кавычек)
