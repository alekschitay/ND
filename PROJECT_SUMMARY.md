# 🎵 Телеграм-бот для мониторинга концертных площадок

## 📋 Описание проекта

Создан полнофункциональный телеграм-бот для автоматического мониторинга сайтов концертных площадок и уведомления пользователей о новых событиях.

## ✅ Реализованные функции

### Основной функционал
- ✅ **Мониторинг веб-страниц** каждые 10 минут
- ✅ **Уведомления только при изменениях** (не спам каждые 10 минут)
- ✅ **Автоматический парсинг событий** с извлечением:
  - Названия события
  - Даты проведения
  - Ссылки на событие
  - Афиши (изображения)
  - Места проведения
  - Цены билетов

### Команды бота
- ✅ `/start` - Приветствие и инструкции
- ✅ `/help` - Подробная справка
- ✅ `/add <ссылка>` - Добавить ссылку для мониторинга
- ✅ `/list` - Показать все отслеживаемые ссылки
- ✅ `/remove <номер>` - Удалить ссылку из мониторинга
- ✅ `/status` - Показать статус мониторинга
- ✅ `/scan` - Принудительно сканировать все ссылки
- ✅ `/scan <номер>` - Сканировать конкретную ссылку
- ✅ `/test <ссылка>` - Протестировать парсинг ссылки
- ✅ `/logs` - Показать последние проверки ссылок

### Технические особенности
- ✅ **Универсальные селекторы** для парсинга различных сайтов
- ✅ **Система кэширования** для определения новых событий
- ✅ **Обработка ошибок** и логирование
- ✅ **Конфигурируемые параметры** через файл config.py
- ✅ **Поддержка множественных пользователей**
- ✅ **Асинхронная обработка** запросов

## 🔧 Настройка

### Основные параметры в config.py:
- `BOT_TOKEN` - Токен телеграм-бота
- `MONITORING_INTERVAL` - Интервал мониторинга (600 сек = 10 мин)
- `MAX_EVENTS_PER_NOTIFICATION` - Максимум событий в уведомлении (5)
- `MAX_IMAGES_PER_NOTIFICATION` - Максимум изображений (3)
- `HTTP_TIMEOUT` - Таймаут HTTP запросов (30 сек)

### Селекторы для парсинга:
- `EVENT_SELECTORS` - Поиск элементов событий
- `TITLE_SELECTORS` - Поиск заголовков
- `DATE_SELECTORS` - Поиск дат
- `VENUE_SELECTORS` - Поиск мест проведения
- `PRICE_SELECTORS` - Поиск цен
