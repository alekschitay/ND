# 🎵 Как открыть Concert Monitor

## ✅ Сервер запущен и работает!

### 🌐 Ссылки для доступа:

1. **Главная страница (рекомендуется):**
   ```
   http://localhost:8000/
   ```

2. **Альтернативная демо-версия:**
   ```
   http://localhost:8080/demo.html
   ```

3. **API endpoints:**
   ```
   http://localhost:8000/api/sites     - Список сайтов
   http://localhost:8000/api/events    - Список событий
   http://localhost:8000/health        - Статус системы
   ```

## 🔧 Если ссылки не работают:

### Вариант 1: Проверьте в терминале
```bash
curl http://localhost:8000/
```

### Вариант 2: Откройте файл напрямую
```bash
# Скопируйте содержимое demo.html в браузер
cat demo.html
```

### Вариант 3: Создайте локальную копию
```bash
# Скачайте файл на свой компьютер
# Откройте demo.html в любом браузере
```

## 📱 Что вы увидите:

- ✅ Красивый интерфейс мониторинга концертных площадок
- ✅ Статистику: 3 сайта, 12 событий, 2 новых
- ✅ Список мониторимых площадок (Крокус, Олимпийский, СК Олимпийский)
- ✅ Последние найденные события с афишами
- ✅ Интерактивные кнопки управления
- ✅ Полную документацию по установке

## 🎯 Основные возможности:

1. **Автоматический мониторинг** - проверка каждые 10 минут
2. **Умные уведомления** - только при новых событиях
3. **Поддержка афиш** - автоматическое извлечение изображений
4. **Гибкие селекторы** - CSS селекторы для любых сайтов
5. **Веб-интерфейс** - простое управление через браузер
