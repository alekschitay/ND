## 📋 **Как использовать:**

### 1. **Добавление сайта для мониторинга**

1. Перейдите на вкладку **"🌐 Сайты"**
2. Нажмите **"➕ Добавить сайт"**
3. Заполните форму:
   - **Название:** Крокус Сити Холл
   - **URL:** https://kroсus-city-hall.ru/events
   - **CSS селектор:** .event-card
   - **Описание:** (необязательно)
4. Нажмите **"✅ Добавить сайт"**

### 2. **Определение CSS селектора**

1. Откройте сайт в браузере
2. Нажмите **F12** (инструменты разработчика)
3. Найдите элемент с событием
4. Щелкните правой кнопкой → **"Копировать"** → **"Копировать селектор"**
5. Вставьте в поле "CSS селектор"

### 3. **Настройка уведомлений**

#### **Telegram:**
1. Перейдите на вкладку **"⚙️ Настройки"**
2. В разделе **"📱 Telegram уведомления"**:
   - **Токен бота:** получите у @BotFather
   - **Chat ID:** ваш ID чата
3. Нажмите **"💾 Сохранить"**

#### **Email:**
1. В разделе **"📧 Email уведомления"**:
   - **SMTP хост:** smtp.gmail.com
   - **SMTP порт:** 587
   - **Email пользователь:** your-email@gmail.com
   - **Email пароль:** ваш пароль приложения
2. Нажмите **"💾 Сохранить"**

### 4. **Управление мониторингом**

1. В разделе **"⏰ Мониторинг"**:
   - **Интервал проверки:** 10 минут (рекомендуется)
   - **Автоматический мониторинг:** включен
2. Нажмите **"💾 Сохранить"**

## 🔧 **Инструкции по настройке:**

### 📱 **Создание Telegram бота:**

1. **Напишите** [@BotFather](https://t.me/botfather)
2. **Отправьте** `/newbot`
3. **Введите название** бота: "Concert Monitor Bot"
4. **Введите username** бота: "concert_monitor_bot"
5. **Сохраните токен** (формат: 123456789:ABCdefGHIjklMNOpqrsTUVwxyz)

### 💬 **Получение Chat ID:**

1. **Напишите боту** `/start`
2. **Перейдите по ссылке:** `https://api.telegram.org/bot<YOUR_BOT_TOKEN>/getUpdates`
3. **Найдите** `"chat":{"id":123456789}` в ответе
4. **Сохраните ID** (число без кавычек)

### 📧 **Настройка Email (Gmail):**

1. **Включите двухфакторную аутентификацию**
2. **Создайте пароль приложения:**
   - Google Account → Security → App passwords
   - Select app: Mail
   - Select device: Other
   - Введите название: Concert Monitor
3. **Используйте пароль приложения** вместо обычного пароля

## 🎯 **Примеры CSS селекторов:**

### **Популярные сайты:**
- **Крокус Сити Холл:** `.event-card`
- **Олимпийский:** `.concert-item`
- **СК Олимпийский:** `.event-block`
- **Крокус Сити Холл:** `.event-item`
- **Москва-Сити:** `.concert-card`

### **Общие селекторы:**
- `.event` - элементы с классом event
- `.concert` - элементы с классом concert
- `[class*="event"]` - элементы содержащие "event" в классе
- `div.event-item` - div элементы с классом event-item
- `li.concert` - li элементы с классом concert
