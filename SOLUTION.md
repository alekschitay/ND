# 🎵 Concert Monitor - РЕШЕНИЕ ПРОБЛЕМЫ

## ❌ Если страница недоступна, вот что делать:

### 📁 **СПОСОБ 1: Откройте файл напрямую (РЕКОМЕНДУЕТСЯ)**

1. **Найдите файл `demo.html`** в папке проекта
2. **Щелкните правой кнопкой мыши** по файлу
3. **Выберите "Открыть с помощью"** → **"Браузер"** (Chrome, Firefox, Safari)
4. **Или просто перетащите** файл `demo.html` в окно браузера

### 💻 **СПОСОБ 2: Через терминал**

```bash
# В папке проекта запустите:
python3 -m http.server 8080

# Затем откройте в браузере:
# http://localhost:8080/demo.html
```

### 📋 **СПОСОБ 3: Скопируйте код**

1. Откройте файл `demo.html` в любом текстовом редакторе
2. Скопируйте весь код (Ctrl+A, Ctrl+C)
3. Создайте новый файл `concert_monitor.html`
4. Вставьте код (Ctrl+V)
5. Сохраните файл
6. Откройте в браузере

### 🌐 **СПОСОБ 4: Проверьте серверы**

```bash
# Проверьте, какие серверы запущены:
ps aux | grep python

# Попробуйте разные порты:
curl http://localhost:3000/
curl http://localhost:8080/
curl http://localhost:8000/
```
