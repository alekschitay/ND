# 🚀 Быстрый запуск телеграм-бота

## ⚡ Запуск за 3 шага

### 1️⃣ Установка зависимостей
```bash
pip3 install python-telegram-bot aiohttp beautifulsoup4 requests lxml
```

### 2️⃣ Запуск бота
```bash
python3 concert_monitor_bot.py
```

### 3️⃣ Использование
1. Найдите бота в Telegram по токену: `7711964415:AAF0tp9uhybhTZ7gPEGLNnpE6TxgvAElYzU`
2. Отправьте `/start`
3. Добавьте ссылку: `/add https://example.com/events`
4. Получайте уведомления! 🎵

## 📱 Команды бота

### Основные команды
- `/start` - Начать работу
- `/add <ссылка>` - Добавить ссылку для мониторинга
- `/list` - Показать отслеживаемые ссылки
- `/remove <номер>` - Удалить ссылку
- `/status` - Статус мониторинга
- `/help` - Справка

### Команды сканирования
- `/scan` - Принудительно сканировать все ссылки
- `/scan <номер>` - Сканировать конкретную ссылку
- `/test <ссылка>` - Протестировать парсинг ссылки
- `/logs` - Показать последние проверки

## 🔗 Примеры ссылок

- `https://example-concert-hall.com/events`
- `https://music-venue.ru/concerts`
- `https://theater.com/afisha`

## ⚙️ Настройка

Отредактируйте `config.py` для изменения:
- Интервала мониторинга (по умолчанию 10 минут)
- Количества событий в уведомлении
- Селекторов для парсинга
