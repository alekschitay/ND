# 🎵 Concert Monitor - Размещение на Tilda

## 📋 Пошаговая инструкция

### 1. **Подготовка контента**

✅ **Файл готов:** `tilda_version.html` - оптимизированная версия для Tilda

### 2. **Создание сайта на Tilda**

1. **Зайдите на** [tilda.cc](https://tilda.cc)
2. **Создайте аккаунт** или войдите в существующий
3. **Нажмите "Создать сайт"**
4. **Выберите шаблон** (любой подходящий)

### 3. **Способы размещения контента**

#### 🌐 **Способ 1: Через Zero Block (рекомендуется)**

1. **Добавьте Zero Block** на страницу
2. **Перейдите в режим редактирования** Zero Block
3. **Нажмите "HTML"** в панели инструментов
4. **Скопируйте код** из файла `tilda_version.html`
5. **Вставьте код** в HTML редактор
6. **Сохраните изменения**

#### 📝 **Способ 2: Через T-блоки**

1. **Добавьте блок "Text"** на страницу
2. **Скопируйте текст** из демо-версии
3. **Вставьте в блок** и отформатируйте
4. **Добавьте изображения** и стили

#### 🎨 **Способ 3: Кастомный дизайн**

1. **Используйте готовые блоки** Tilda
2. **Создайте Hero-секцию** с градиентом
3. **Добавьте блоки статистики** (4 карточки)
4. **Создайте секцию возможностей** (6 карточек)
5. **Добавьте блоки сайтов** и событий

### 4. **Оптимизация для Tilda**

#### ✅ **Что уже сделано:**
- Убраны внешние зависимости (Bootstrap, Font Awesome)
- Использованы только встроенные CSS стили
- Адаптивный дизайн для мобильных устройств
- Оптимизированная структура HTML

#### 🔧 **Дополнительные настройки:**

1. **SEO оптимизация:**
   - Добавьте мета-теги в настройках страницы
   - Укажите заголовок: "Concert Monitor - Мониторинг концертных площадок"
   - Добавьте описание: "Автоматический мониторинг сайтов концертных площадок с уведомлениями о новых событиях"

2. **Настройка домена:**
   - Подключите свой домен в настройках сайта
   - Настройте SSL сертификат

3. **Аналитика:**
   - Подключите Google Analytics
   - Настройте Яндекс.Метрику

### 5. **Структура контента для Tilda**

#### 🎯 **Hero-секция:**
```
Заголовок: Concert Monitor
Подзаголовок: Автоматический мониторинг сайтов концертных площадок
Описание: Отслеживает изменения и присылает уведомления только при появлении новых событий
```

#### 📊 **Статистика (4 блока):**
- 3 мониторимых сайта
- 12 всего событий  
- 2 новых события
- 3 активных сайта

#### 🌟 **Возможности (6 блоков):**
1. Автоматический мониторинг
2. Умные уведомления
3. Поддержка афиш
4. Гибкие селекторы
5. Статистика и история
6. Простое управление

#### 🌐 **Мониторимые сайты:**
- Крокус Сити Холл
- Олимпийский
- СК Олимпийский

#### 🎵 **События:**
- Концерт группы 'Ария' (НОВОЕ)
- Рок-фестиваль 'Нашествие' (НОВОЕ)
- Концерт 'Сплин'

### 6. **Публикация**

1. **Нажмите "Опубликовать"** в редакторе Tilda
2. **Выберите тарифный план** (если нужен)
3. **Настройте домен** (если есть)
4. **Проверьте сайт** на мобильных устройствах
