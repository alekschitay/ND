Установка и настройка

🐳 С Docker (рекомендуется)

# Клонирование репозитория git clone <repository-url> cd concert-monitor # Настройка переменных окружения cp .env.example .env # Отредактируйте .env файл # Запуск с Docker Compose docker-compose up -d

💻 Локальная установка

# Установка зависимостей pip install -r requirements.txt # Настройка базы данных python init_db.py # Запуск приложения python run.py

📱 Настройка уведомлений

Telegram:

  1. Создайте бота через @BotFather
  2. Получите токен бота
  3. Узнайте ваш Chat ID
  4. Добавьте токен и Chat ID в .env файл

Email: Настройте SMTP параметры в .env файле
