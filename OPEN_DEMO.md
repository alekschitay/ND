# 🎵 Concert Monitor - Как открыть демо

## ❌ Если ссылки не работают, вот что делать:

### 📁 Способ 1: Откройте файл напрямую
1. Найдите файл `demo.html` в папке проекта
2. Щелкните по нему правой кнопкой мыши
3. Выберите "Открыть с помощью" → "Браузер" (Chrome, Firefox, Safari)
4. Или просто перетащите файл в окно браузера

### 🌐 Способ 2: Скопируйте содержимое
1. Откройте файл `demo.html` в любом текстовом редакторе
2. Скопируйте весь код (Ctrl+A, Ctrl+C)
3. Откройте браузер
4. Нажмите F12 (открыть инструменты разработчика)
5. Вставьте код в консоль или создайте новый HTML файл

### 💻 Способ 3: Через терминал
```bash
# Откройте терминал в папке проекта
python3 -m http.server 8080

# Затем откройте в браузере:
# http://localhost:8080/demo.html
```
